# Kevin Luo Personal Site

A minimal personal website built with Next.js, TypeScript, Tailwind CSS, and Markdown content.

## Local Development

```bash
npm install
npm run dev
```

Open http://127.0.0.1:3000.

## Structure

- `src/app` contains the routes and page layouts.
- `src/components` contains reusable UI components.
- `src/lib` contains site data, project data, and content helpers.
- `content/posts` contains Markdown blog posts.
- `public` contains static assets.
